## 四、亏损公式

### 最容易亏的参数组合

```
连续天数=6 + 涨幅限制≥15% + 日止盈=2次 + 高低点窗口=5根 + 高低点最少个数=2个
```

### 最安全的参数组合

```
连续天数=2/4天 + 涨幅限制=5% + 日止盈=1/3次 + 高低点窗口=3根 + 高低点最少个数=3个
```

## 五、核心结论

| 参数 | 最佳取值 | 最差取值 |
|------|---------|---------|
| 连续天数 | 2天或4天 | 6天 |
| 涨幅限制 | 5% | ≥15% |
| 日止盈次数 | 4次或5次 | 2次 |
| 高低点窗口 | 3根 | 5根 |
| 高低点最少个数 | 3个 | 2个 |

> **亏损不是策略问题，是参数选择问题。现货永不亏钱，合约参数选错就爆仓。**
